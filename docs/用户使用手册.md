## 软件简介

**AICowork** 是一款智能 AI 协作助手，支持多种 AI 服务商，具备记忆系统、技能扩展等功能，帮助您在办公、学习、创作等场景提高效率。

### 主要特点

- ✅ 支持 60+ AI 服务商（智谱 AI、DeepSeek、OpenAI 等）
- ✅ 本地记忆系统，AI 能记住重要信息
- ✅ 自定义技能扩展
- ✅ MCP 服务器集成
- ✅ 多语言界面支持
- ✅ 数据本地存储，保护隐私

## 快速开始

### 第一步：安装软件

**Windows 用户：**
1. 下载 `AICowork-win32-x64-0.1.0.exe`
2. 双击运行安装程序
3. 按提示完成安装

**macOS 用户：**
1. 下载对应芯片版本的 `.dmg` 文件（ARM 或 Intel）
2. 打开 DMG 文件
3. 将 AICowork 拖拽到应用程序文件夹

**Linux 用户：**
1. 下载 `AICowork-0.1.0.AppImage`
2. 添加执行权限：`chmod +x AICowork-0.1.0.AppImage`
3. 双击运行

### 第二步：获取 API Key

首次使用需要配置 AI 服务：

1. 选择一个 AI 服务商（推荐以下之一）：
   - **智谱 AI**：https://open.bigmodel.cn/
   - **DeepSeek**：https://platform.deepseek.com/
   - **OpenAI**：https://platform.openai.com/

2. 注册账号并登录

3. 进入控制台，创建 API Key

4. 复制生成的 API Key（格式类似：`sk-xxxxxxxxxxxxx`）

### 第三步：配置 API

软件首次启动时会自动跳转到设置页面：

1. 点击左侧菜单的 **"API 配置"**

2. 填写配置信息：
   - **配置名称**：给这个配置起个名字（如"智谱 AI 配置"）
   - **API Key**：粘贴刚才复制的密钥
   - **API 厂商**：选择对应的服务商
   - **基础 URL**：通常自动填充，无需修改
   - **模型名称**：选择要使用的模型

3. 点击 **"保存配置"** 按钮

4. 点击配置右侧的 **"设为激活"** 按钮

5. 点击 **"测试连接"** 确认配置正确

### 第四步：开始对话

1. 返回主界面（点击左上角 logo 或关闭设置页）

2. 点击左侧 **"+ 新对话"** 按钮

3. 在底部输入框输入您的问题

4. 按 **Enter 键** 发送

5. 等待 AI 回复

## 基础功能

### 1. AI 对话

#### 创建新对话

- **方法一**：点击左侧栏的 **"+ 新对话"** 按钮
- **方法二**：使用快捷键 `Ctrl/Cmd + N`

#### 进行多轮对话

1. 发送第一条消息后，AI 会回复
2. 继续在输入框输入问题
3. AI 会记住上下文，进行连续对话

#### 切换历史对话

- 点击左侧栏的历史会话卡片
- 自动加载该会话的所有对话内容

#### 重命名会话

1. 在左侧栏找到要重命名的会话
2. 鼠标右键点击会话卡片
3. 选择 **"重命名"**
4. 输入新名称并确认

#### 删除会话

1. 在左侧栏找到要删除的会话
2. 鼠标右键点击会话卡片
3. 选择 **"删除"**
4. 确认删除操作

### 2. 输入增强功能

#### 使用斜杠命令

在输入框输入 `/` 可查看所有可用命令：

| 命令 | 功能 | 使用示例 |
|------|------|----------|
| `/plan` | 制定实施计划 | `/plan` 制定项目计划 |
| `/help` | 显示帮助信息 | `/help` |
| `/new` | 新建会话 | `/new` |
| `/sessions` | 会话管理 | `/sessions` |
| `/memory` | 记忆管理 | `/memory` |
| `/settings` | 打开设置 | `/settings` |

#### 输入多行内容

按 `Shift + Enter` 换行，输入多行文本后按 `Enter` 发送。

#### 清空输入

点击输入框右侧的 **"×"** 按钮清空当前输入。

### 3. 消息操作

#### 查看消息详情

每条消息卡片包含：
- **发送者**：用户（右侧）或 AI（左侧）
- **时间戳**：消息发送时间
- **内容**：支持 Markdown 格式渲染

#### 复制消息内容

1. 点击消息卡片
2. 选择要复制的文本
3. 按 `Ctrl/Cmd + C` 复制

#### 滚动查看历史

- 滚动到顶部自动加载更多历史消息
- 底部出现 **"新消息"** 按钮时点击跳转最新

## 高级功能

### 1. 记忆系统深度使用

#### 什么内容应该存储到记忆？

**应该存储：**
- ✅ 项目技术决策和架构选择
- ✅ 用户明确的偏好设置和工作习惯
- ✅ 项目特定的约定和规范
- ✅ 复杂问题的解决方案（经过提炼）
- ✅ 常用文档模板

**不应该存储：**
- ❌ 临时性对话和闲聊
- ❌ 重复的信息
- ❌ 过于琐碎的细节
- ❌ 原始输出日志

| 标签 | 用途 | 示例 |
|------|------|------|
| `project` | 项目相关信息 | 技术栈、架构决策 |

| `technical` | 技术知识 | 命令用法、配置方法 |
| `context` | 上下文信息 | 项目背景、业务逻辑 |
| `custom` | 自定义分类 | 用户自定义分类 |

## 常见问题

### Q1: 提示"API Key 未配置"

**解决方法：**
1. 点击设置 → API 配置
2. 添加新的 API 配置
3. 保存并设为激活

### Q2: 发送消息无响应

**可能原因：**
- 网络连接问题
- API Key 无效
- API 服务商服务异常

**解决方法：**
1. 检查网络连接
2. 测试 API 连接（设置 → API 配置 → 测试连接）
3. 检查 API Key 是否正确

### Q3: 如何删除历史对话

**方法：**
1. 在左侧栏找到要删除的会话
2. 右键点击会话卡片
3. 选择"删除"
4. 确认删除

### Q4: 记忆系统如何使用

**最佳实践：**
1. 设置 → 记忆系统
2. 添加新记忆
3. 填写标题和内容
4. 选择合适标签
5. 保存后 AI 会自动调用

### Q5: 如何使用技能

**方法：**
1. 创建技能（设置 → 技能管理）
2. 在对话中输入技能名称（如 `/my-skill`）
3. AI 执行技能定义的操作

### Q6: 数据存储在哪里

**Windows:**
```
C:\Users\你的用户名\AppData\Roaming\AICowork\
```

**macOS:**
```
~/Library/Application Support/AICowork/
```

**Linux:**
```
~/.config/AICowork/
```

### Q7: 如何备份配置

**方法：**
1. 进入配置目录（见上方路径）
2. 复制整个 `AICowork` 文件夹
3. 保存到安全位置

恢复时直接替换文件夹即可。

### Q8: 软件占用空间大怎么办

**解决方法：**
1. 删除旧的会话（会话恢复）
2. 清理不需要的记忆
3. 定期清理日志文件

### Q9: 如何升级软件

**方法：**
1. 下载最新版本安装包
2. 覆盖安装（配置会保留）
3. 或解压到新目录后复制配置文件夹

### Q10: AI 回复质量差怎么办

**优化建议：**
1. 调整温度参数（设置 → 代理管理）
2. 添加相关记忆（记忆系统）
3. 使用技能扩展 AI 能力
4. 更换更强大的模型
