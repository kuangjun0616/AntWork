## 快速开始

### 什么是 AICowork？

**AICowork** 是一款安装在电脑上的 AI 智能助手，可以帮你：

- ✍️ **写文档** - 写报告、写总结、写邮件
- 💻 **写代码** - 生成代码、解释代码、审查代码
- 📚 **学习助手** - 解释概念、回答问题、制定学习计划
- 🎨 **创意工作** - 头脑风暴、内容策划、创意生成
- 📊 **数据分析** - 分析数据、生成报告、趋势预测

### 系统要求

你的电脑需要满足以下要求：

| 电脑类型 | 要求 |
|---------|------|
| **Windows** | Windows 10 或更高版本 |
| **苹果 Mac** | macOS 11 (Big Sur) 或更高版本 |
| **Linux** | Ubuntu 20.04 或同等发行版 |
| **内存** | 建议 8GB 及以上 |
| **硬盘空间** | 至少 500MB 可用空间 |

## 安装应用

### 步骤 1: 下载安装包

#### Windows 用户

1. 打开浏览器，访问：https://github.com/Pan519/AICowork/releases
2. 找到最新版本（如 v0.1.0）
3. 下载文件：`AICowork-win32-x64-0.1.0.exe`
4. 等待下载完成（约 150MB）

#### 苹果 Mac 用户

1. 打开浏览器，访问：https://github.com/Pan519/AICowork/releases
2. 找到最新版本（如 v0.1.0）
3. 根据你的 Mac 芯片选择下载：
   - **M1/M2/M3 芯片**：下载 `AICowork-0.1.0-arm64.dmg`
   - **Intel 芯片**：下载 `AICowork-0.1.0-x64.dmg`
4. 等待下载完成（约 160MB）

#### Linux 用户

1. 打开浏览器，访问：https://github.com/Pan519/AICowork/releases
2. 找到最新版本（如 v0.1.0）
3. 下载文件：`AICowork-0.1.0.AppImage`
4. 等待下载完成（约 150MB）

---

### 步骤 2: 安装应用

#### Windows 安装步骤

1. 找到下载的文件：`AICowork-win32-x64-0.1.0.exe`
2. **双击** 该文件
3. 如果弹出安全提示，点击"更多信息" → "仍要运行"
4. 在安装向导中，点击"下一步"
5. 选择安装位置（默认即可），点击"下一步"
6. 点击"安装"
7. 等待安装完成
8. 点击"完成"，应用自动启动

**📌 小贴士**：安装完成后，可以在桌面找到 AICowork 的快捷方式图标

#### 苹果 Mac 安装步骤

1. 找到下载的文件：`AICowork-0.1.0-xxx.dmg`
2. **双击** 该文件（会打开一个虚拟磁盘窗口）
3. 将 **AICowork 图标拖拽** 到"应用程序"文件夹
4. 等待复制完成
5. 在"启动台"中找到 AICowork，点击启动
6. 如果提示"来自身份不明的开发者"，右键点击 → "打开" → "打开"

**📌 小贴士**：可以将 AICowork 固定到程序坞，方便下次使用

#### Linux 安装步骤

1. 打开终端（按 `Ctrl + Alt + T`）
2. 进入下载文件所在目录：
   ```bash
   cd ~/Downloads
   ```
3. 给文件添加执行权限：
   ```bash
   chmod +x AICowork-0.1.0.AppImage
   ```
4. 运行应用：
   ```bash
   ./AICowork-0.1.0.AppImage
   ```

**📌 小贴士**：可以创建桌面快捷方式，方便下次启动

---

### 步骤 3: 首次启动

1. 应用启动后，会看到主界面
2. 界面中间会提示：**"请先配置 API Key"**
3. 点击"配置 API Key"按钮，进入设置页面
4. 按照下一章节的步骤配置 API Key

## 获取 API Key

### 什么是 API Key？

**API Key** 就像是你使用 AI 服务的"钥匙"。你需要从 AI 服务商那里获取这把"钥匙"，才能使用他们的 AI 服务。

### 推荐的 AI 服务商

我们推荐以下几家提供免费额度的服务商：

| 服务商 | 免费额度 | 优点 | 获取地址 |
|-------|---------|------|---------|
| **智谱 AI** | 新用户 25 元 | 中文能力强，响应快 | [点击跳转](https://open.bigmodel.cn) |
| **DeepSeek** | 每天 100 万 tokens | 性价比高，适合大量使用 | [点击跳转](https://platform.deepseek.com) |
| **通义千问** | 新用户有额度 | 阿里云出品，稳定可靠 | [点击跳转](https://tongyi.aliyun.com) |
| **OpenAI** | 需付费 | 世界上最强大的 AI | [点击跳转](https://platform.openai.com) |

**💡 建议**：新手推荐先从"智谱 AI"或"DeepSeek"开始，因为有免费额度。

---

### 详细获取步骤（以智谱 AI 为例）

#### 步骤 1: 注册账号

1. 打开浏览器，访问：https://open.bigmodel.cn
2. 点击右上角"注册"按钮
3. 选择注册方式：
   - **手机号注册**（推荐）
   - **微信扫码注册**
4. 填写手机号，获取验证码
5. 设置密码（至少 8 位，包含字母和数字）
6. 点击"注册"
7. 完成手机号验证

#### 步骤 2: 实名认证（可选，但建议完成）

1. 登录后，点击右上角"用户中心"
2. 选择"实名认证"
3. 填写真实姓名和身份证号
4. 完成人脸识别（需要在手机上操作）
5. 认证成功后，会获得 **25 元** 免费额度

**📌 小贴士**：实名认证可以获得更多免费额度，不认证也可以使用，但额度较少。

#### 步骤 3: 创建 API Key

1. 登录后，点击右上角"用户中心"
2. 在左侧菜单中，选择"API Key"
3. 点击右上角"创建新的 API Key"
4. 弹出对话框，输入备注名称（如："我的电脑"）
5. 点击"确认"
6. **重要**：复制显示的 API Key（格式类似：`5f3a2b1c4d5e6f7g8h9i0j1k2l3m4n5o6p7q8r9s0t`）
7. 立即保存到安全的地方（如记事本）

**⚠️ 重要提示**：
- API Key 只显示一次，请务必立即复制保存
- 不要泄露给你的 API Key
- 建议保存到手机备忘录或密码管理器

## 配置应用

### 步骤 1: 打开设置页面

1. 启动 AICowork 应用
2. 点击左下角的 **⚙️ 设置图标**
3. 进入设置页面

---

### 步骤 2: 进入 API 配置

1. 在设置页面左侧，点击 **"API 配置"**
2. 右侧显示"API 配置管理"页面
3. 点击 **"添加新配置"** 按钮

---

### 步骤 3: 填写配置信息

弹出"添加 API 配置"对话框，填写以下信息：

#### ① 配置名称

- **说明**：给这个配置起个名字，方便识别
- **示例**：`智谱 AI`、`DeepSeek`、`我的主要 AI`
- **填写**：输入 `智谱 AI`（推荐使用服务商名称）

#### ② API Key

- **说明**：粘贴你刚才复制的 API Key
- **填写**：
  1. 在输入框中点击一下
  2. 按 `Ctrl + V`（Windows/Linux）或 `Cmd + V`（Mac）粘贴
  3. 确认 Key 完整（不要有空格或换行）

#### ③ API 厂商

- **说明**：选择你使用的 AI 服务商
- **操作**：
  1. 点击下拉菜单
  2. 选择 `zhipu`（智谱 AI）
  3. 其他服务商对应：
     - DeepSeek → `deepseek`
     - OpenAI → `openai`
     - 通义千问 → `qwen`

#### ④ 模型名称

- **说明**：选择使用的 AI 模型
- **操作**：
  1. 点击下拉菜单
  2. 选择 `glm-4-flash`（推荐，速度快）
  3. 其他选项：
     - `glm-4` - 标准模型
     - `glm-4-plus` - 增强模型（消耗更多额度）

---

### 步骤 4: 保存配置

1. 检查填写的信息是否正确
2. 点击对话框底部的 **"保存配置"** 按钮
3. 配置列表中出现你刚添加的配置
4. 此时配置处于未激活状态（灰色）

---

### 步骤 5: 激活配置

1. 在配置列表中，找到你刚添加的配置
2. 点击配置右侧的 **"设为激活"** 按钮
3. 按钮变为绿色，显示"✓ 已激活"
4. **完成**！现在可以使用 AI 对话了

---

### 步骤 6: 测试连接（可选）

1. 在配置列表中，点击 **"测试连接"** 按钮
2. 等待几秒钟
3. 如果显示 **"✓ 连接成功"**，说明配置正确
4. 如果显示 **"✗ 连接失败"**，请检查：
   - API Key 是否正确
   - 网络连接是否正常
   - 服务商是否有可用额度

---

## 开始对话

### 步骤 1: 创建新对话

1. 点击左侧边栏的 **"+ 新对话"** 按钮
2. 或点击左上角 **"对话"** 标签
3. 主界面出现聊天窗口

---

### 步骤 2: 发送第一条消息

1. 在底部输入框中输入你的问题
2. 示例问题：
   - `你好，请介绍一下你自己`
   - `帮我写一封请假条`
   - `解释什么是人工智能`
   - `帮我想一个生日祝福语`
3. 按 **Enter** 键发送（或点击右侧的"发送"按钮）

---

### 步骤 3: 查看 AI 回复

1. AI 会立即开始回复
2. 回复内容会以打字效果逐字显示
3. 等待 AI 回复完成（通常几秒钟）

---

### 步骤 4: 继续对话

1. 在输入框中输入第二个问题
2. AI 会记住之前的对话内容
3. 支持多轮连续对话

---

### 对话示例

#### 示例 1: 写请假条

**你**: `帮我写一封病假条，理由是感冒，需要请假两天`

**AI**: 会生成一份完整的请假条，包含：
- 称呼
- 请假原因
- 请假时间
- 恳请批准
- 署名和日期

#### 示例 2: 学习知识

**你**: `请解释什么是递归，并用一个简单的例子说明`

**AI**: 会详细解释：
- 递归的概念
- 使用生活例子类比
- 提供代码示例
- 解释优缺点

#### 示例 3: 创意工作

**你**: `帮我想 10 个有创意的咖啡店名字`

**AI**: 会列出 10 个创意名字，并附上每个名字的含义和寓意。

## 使用记忆系统

### 什么是记忆系统？

**记忆系统** 就像是 AI 的"笔记本"，可以让 AI 记住重要信息，以后对话时会自动调用。

### 记忆系统的用途

- 📝 **记住项目信息** - 技术栈、架构设计、业务逻辑
- 👤 **记住个人偏好** - 语言风格、格式习惯、常用设置
- 📚 **积累知识文档** - 学习笔记、教程要点、最佳实践
- 💼 **存储上下文** - 项目背景、业务规则、团队约定

---

### 步骤 1: 打开记忆系统

1. 点击左下角的 **⚙️ 设置图标**
2. 在左侧菜单中，点击 **"记忆系统"**
3. 右侧显示"记忆管理"页面

---

### 步骤 2: 添加新记忆

1. 点击 **"添加新记忆"** 按钮
2. 弹出"添加记忆"对话框

---

### 步骤 3: 填写记忆内容

#### ① 标题

- **说明**：简短描述这条记忆
- **示例**：`我的工作项目`、`常用的写作风格`、`Python 学习笔记`
- **填写**：输入 `我的工作项目`

#### ② 内容

- **说明**：详细的记忆内容
- **示例**：
  ```
  我的工作项目是一个基于 React + TypeScript 的前端项目。
  技术栈包括：React 19, TypeScript 5.9, Tailwind CSS, Zustand。
  项目使用 Electron 构建桌面应用。
  代码规范：使用 ESLint，遵循 Airbnb 风格指南。
  ```
- **填写**：输入你想让 AI 记住的内容

#### ③ 标签分类

选择合适的标签，方便管理：

| 标签 | 用途 | 示例 |
|-----|------|------|
| **project** | 项目信息 | 技术栈、架构、业务逻辑 |
| **preference** | 个人偏好 | 语言风格、格式习惯 |
| **technical** | 技术知识 | 学习笔记、教程要点 |
| **context** | 上下文信息 | 项目背景、业务规则 |
| **custom** | 自定义 | 其他内容 |

- **操作**：点击下拉菜单，选择合适的标签
- **填写**：选择 `project`（项目信息）

---

### 步骤 4: 保存记忆

1. 检查填写的内容
2. 点击 **"保存"** 按钮
3. 记忆列表中出现新添加的记忆
4. **完成**！AI 已经记住了这条信息

---

### 步骤 5: 验证记忆是否生效

1. 创建新对话
2. 输入问题：`我的工作项目使用什么技术栈？`
3. AI 会从记忆系统中检索相关信息并回答
4. 如果 AI 正确回答了，说明记忆已生效

---

## 使用技能系统

### 什么是技能系统？

**技能系统** 就像是 AI 的"工具箱"，每个技能都是一个特定的功能或能力。

你可以：
- 🔧 **使用现有技能** - 执行特定任务
- 📝 **创建自定义技能** - 定义自己的工作流程

---

### 使用技能的两种方式

#### 方式一：使用 `/input` 命令

在聊天输入框中输入：

```
/input 技能名称 --参数1="值1" --参数2="值2"
```

**示例**：
```
/input translator --target="English" --text="你好，世界"
```

#### 方式二：直接对话

直接告诉 AI 你想做什么：

```
你：帮我翻译成英文：你好，世界
```

---

### 创建自定义技能

#### 步骤 1: 找到技能目录

1. 打开文件管理器
2. 在地址栏输入以下路径（根据你的系统）：

**Windows**:
```
C:\Users\你的用户名\.claude\skills
```

**Mac**:
```
~/.claude/skills
```

**Linux**:
```
~/.claude/skills
```

3. 如果目录不存在，手动创建

---

#### 步骤 2: 创建技能文件

1. 在 `skills` 目录中，创建一个新的文本文件

## 常见问题

### Q1: 提示"API Key 未配置"怎么办？

**原因**：还没有添加 API 配置

**解决步骤**：
1. 点击左下角"设置"图标
2. 选择"API 配置"
3. 点击"添加新配置"
4. 填写配置信息（详见"配置应用"章节）
5. 保存并设为激活

---

### Q2: 发送消息后没有反应？

**可能原因**：

#### 原因 1: API Key 错误

**检查方法**：
1. 打开设置 → API 配置
2. 点击"测试连接"
3. 如果显示"连接失败"，说明 Key 有误

**解决方法**：
- 重新复制 API Key
- 删除旧配置，重新添加
- 确认没有多余的空格或换行

---

#### 原因 2: 网络连接问题

**检查方法**：
1. 打开浏览器，访问其他网站
2. 如果无法访问，说明网络有问题

**解决方法**：
- 检查网络连接
- 尝试切换网络（如使用手机热点）
- 检查防火墙设置

---

#### 原因 3: 服务商额度用尽

**检查方法**：
1. 登录 AI 服务商的控制台
2. 查看"用量统计"或"余额"
3. 确认是否还有可用额度

**解决方法**：
- 充值或购买额度
- 或更换其他有免费额度的服务商

---

### Q3: AI 回复很慢怎么办？

**可能原因**：

#### 原因 1: 模型选择问题

**解决方法**：
- 切换到更快的模型（如 `glm-4-flash`）
- 在设置中，编辑 API 配置
- 选择"轻量级"或"快速"模型

#### 原因 2: 网络延迟

**解决方法**：
- 切换到更快的网络
- 尝试使用 VPN（如果服务商在海外）

#### 原因 3: 问题太复杂

**解决方法**：
- 将复杂问题拆分成多个简单问题
- 逐个提问

---

### Q4: 如何删除历史对话？

**方法一：单个删除**

1. 在左侧边栏中，找到要删除的会话
2. **右键点击**该会话
3. 选择"删除"
4. 确认删除

**方法二：批量删除**

1. 点击左下角"设置"
2. 选择"会话管理"
3. 勾选要删除的会话
4. 点击"批量删除"

---

### Q5: 如何导出对话记录？

**步骤**：

1. 打开要导出的会话
2. 点击右上角的"..."菜单
3. 选择"导出对话"
4. 选择导出格式：
   - **Markdown** - 适合阅读和编辑
   - **PDF** - 适合打印和分享
   - **纯文本** - 适合存档
5. 选择保存位置
6. 点击"保存"

---

### Q6: 记忆系统不生效？

**检查步骤**：

1. 确认记忆已保存
   - 打开设置 → 记忆系统
   - 查看记忆列表
   - 确认你想让 AI 记住的内容在列表中

2. 测试记忆是否生效
   - 创建新对话
   - 输入与记忆相关的问题
   - 查看是否调用记忆

3. 如果仍不生效
   - 尝试重启应用
   - 检查记忆内容是否清晰明确
   - 使用更具体的关键词

---

### Q7: 如何更换 API Key？

**步骤**：

1. 打开设置 → API 配置
2. 找到要更换的配置
3. 点击"编辑"按钮
4. 删除旧的 API Key
5. 粘贴新的 API Key
6. 点击"保存"
7. 点击"测试连接"验证

---

### Q8: 可以同时使用多个 AI 服务商吗？

**答案**：可以！

**步骤**：

1. 添加多个 API 配置
   - 添加"智谱 AI"配置
   - 添加"DeepSeek"配置
   - 添加"通义千问"配置

2. 切换激活配置
   - 在设置中，点击"设为激活"
   - 随时切换不同的 AI 服务商

3. 根据需求选择
   - 智谱 AI - 中文能力强
   - DeepSeek - 性价比高
   - OpenAI - 能力最强

---

### Q9: 应用占用空间太大怎么办？

**原因**：主要是日志文件和数据库文件

**解决方法**：

#### 清理日志文件

1. 打开应用数据目录：
   - **Windows**: `C:\Users\你的用户名\AppData\Roaming\AICowork\logs`
   - **Mac**: `~/Library/Application Support/AICowork/logs`
   - **Linux**: `~/.config/AICowork/logs`

2. 删除旧的日志文件（保留最近的）

#### 清理历史会话

1. 打开会话管理
2. 删除不需要的历史会话
3. 数据库会自动缩小

---

### Q10: 如何备份数据？

**步骤**：

1. 关闭 AICowork 应用

2. 找到应用数据目录：
   - **Windows**: `C:\Users\你的用户名\AppData\Roaming\AICowork`
   - **Mac**: `~/Library/Application Support/AICowork`
   - **Linux**: `~/.config/AICowork`

3. 复制整个 `AICowork` 文件夹到安全位置：
   - U 盘
   - 移动硬盘
   - 云盘（如百度网盘、OneDrive）

4. 恢复数据时，将备份文件夹复制回原位置即可
