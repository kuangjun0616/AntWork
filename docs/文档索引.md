## 🚀 快速导航

### 我想...

**开始使用 AICowork**
1. 阅读 [用户使用手册 - 快速开始](./用户使用手册.md#快速开始)
2. 配置 API 密钥
3. 开始对话

**开发新功能**
1. 阅读 [开发者文档 - 开发环境搭建](./开发者文档.md#开发环境搭建)
2. 了解 [项目结构](./开发者文档.md#项目结构)
3. 查看 [核心模块说明](./开发者文档.md#核心模块说明)

**创建插件**
1. 阅读 [插件开发指南](./插件开发指南.md)
2. 选择插件类型（技能/MCP/代理）
3. 查看开发示例

**调试问题**
1. 查看 [用户使用手册 - 常见问题](./用户使用手册.md#常见问题)
2. 查看 [开发者文档 - 常见问题](./开发者文档.md#常见问题)
3. 查看日志文件

**了解架构**
1. 阅读 [架构设计文档 - 系统概述](./架构设计文档.md#系统概述)
2. 查看 [分层架构](./架构设计文档.md#分层架构)
3. 了解 [数据流设计](./架构设计文档.md#数据流设计)

**调用 API**
1. 查看 [API 参考文档 - IPC API](./API参考文档.md#ipc-api)
2. 查看 [类型定义](./API参考文档.md#类型定义)
3. 查看使用示例
